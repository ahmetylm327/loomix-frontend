# 🚀 Loomix ERP - Frontend

Loomix, işletmelerin personel, üretim ve stok süreçlerini dijitalleştirmek için geliştirilmiş modern bir **ERP (Kurumsal Kaynak Planlama)** arayüzüdür. Bu proje, React.js kullanılarak geliştirilmiş ve bulut ortamında (Render) canlıya alınmıştır.

## 🌐 Canlı Uygulama
Uygulamaya şu adresten ulaşabilirsiniz: 
👉 [https://loomix-panel.onrender.com](https://loomix-panel.onrender.com)

## ✨ Öne Çıkan Özellikler
- **Personel Yönetimi (CRUD):** Personel ekleme, silme ve güncelleme işlemleri.
- **Üretim Takibi:** Günlük üretim verilerinin girişi ve stok entegrasyonu.
- **Yapay Zeka Destekli Tahmin:** Backend'den gelen verilerle gelecek dönem üretim öngörüleri.
- **Responsive Tasarım:** Mobil ve tablet cihazlarla tam uyumlu arayüz.
- **Güvenli Giriş:** JWT tabanlı kimlik doğrulama sistemi.

## 🛠️ Teknik Yığın (Tech Stack)
- **Frontend:** React.js
- **Stil:** Tailwind CSS / CSS3
- **HTTP İstekleri:** Axios
- **Yönlendirme:** React Router DOM v6
- **Deployment:** Render
